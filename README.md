# Generador de Arte ASCII Animado

Proyecto de programación que genera arte ASCII y animaciones en la terminal usando Python.

## Descripción

Este programa permite generar diferentes tipos de arte ASCII desde un menú interactivo.

El usuario puede:

- Crear patrones geométricos
- Generar banners con texto
- Crear marcos decorativos
- Ver animaciones simples
- Generar tablas de multiplicar visuales

## Cómo ejecutar

1. Descargar el repositorio
2. Ejecutar el archivo:

python arte_ascii_equipo.py
